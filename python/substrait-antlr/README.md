# substrait-antlr

Substrait ANTLR parser and grammar utilities for Python.

## Installation

```bash
pip install substrait-antlr
```

## Usage

```python
from substrait_antlr import ...
```

## Development

Install development dependencies:

```bash
pip install -e ".[dev]"
```

Run tests:

```bash
pytest
```
